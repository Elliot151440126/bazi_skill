# 八字命理分析 Skill

一个基于 Claude Code 的八字命理分析 Skill。

## 安装

```bash
# 克隆到 Claude Code 的 skills 目录
git clone git@github.com:ElliotWillBeRich/bazi_skill.git ~/.agents/skills/bazi
```

安装后在 Claude Code 中直接输入四柱八字即可触发：

```
帮我看看八字，男，年柱甲子 月柱丙寅 日柱庚午 时柱戊寅
```

```
分析八字：女，1985年生，乙丑年 己卯月 丙寅日 戊子时
```

> ⚠️ Skill 不自行排盘（万年历换算容易出错）。如果还没排过，skill 会引导你用排盘工具排好后再分析。

无需额外配置，Claude Code 会自动加载 `SKILL.md` 作为分析指引。

## 功能特点

- **十神配置**：以日干为准自动配十神
- **干支互动分析**：穷举合、冲、刑、害、破等 9 类关系
- **格局判断**：正格八格、特殊格局
- **用神取用**：判断用神、忌神、喜神、仇神
- **前事校准**：推断过去 3 年事件供验证，反馈后自动修正
- **大运流年流月**：逐层推演，精确定位事件应期

## 分析顺序

**严格遵循以下顺序（铁律）：**

```
排盘 → 天干地支互动 → 原局（先天格局）→ 前事校准 → 大运（十年方向）→ 流年（年份吉凶）→ 流月（月份应期）
```

### 比喻理解

| 层次 | 作用 | 比喻 |
|------|------|------|
| **原局** | 先天格局、一生框架 | 剧本 |
| **大运** | 十年大方向 | 人生章节 |
| **流年** | 具体年份吉凶 | 章节中的情节 |
| **流月** | 精确到月 | 具体场景 |

## 使用方法

### 输入信息
```
"帮我看看八字，男，1990年8月15日下午3点"
"分析八字：女，农历1985年五月初八子时"
"看下今年运势，1992年3月20日卯时生，男"
```

### 必须信息
- 性别（男/女）
- 出生年、月、日、时
- 如为农历，需说明

## 输出内容

1. **命主信息**：八字四柱、十神分布
2. **天干地支互动**：合、冲、刑、害、破关系穷举
3. **原局分析**：五行力量、日主旺衰、格局、用神忌神
4. **前事校准**：推断过去 3 年事件，用户确认后再继续深度分析
5. **大运分析**：大运排列、当前大运详细分析
6. **流年分析**：当年流年吉凶、与大运原局关系
7. **流月分析**：流月排列、重点月份、事件应期
8. **综合建议**：基于以上分析的建议

## 核心规则

### 判断日主旺衰
1. **看月令**（最重要，占40-50%力量）
2. **看通根**（地支根气）
3. **看天干生扶**
4. **综合判断**：身强/身弱/从强/从弱/中和

### 取用神原则
- **身强**：取克泄耗为用（官杀、食伤、财星）
- **身弱**：取生扶为用（印星、比劫）
- **从强**：顺其旺势，取印比为用
- **从弱**：顺其弱势，取克泄耗为用

### 大运顺逆
| 年干阴阳 | 性别 | 大运方向 |
|----------|------|----------|
| 阳（甲丙戊庚壬） | 男 | 顺排 |
| 阴（乙丁己辛癸） | 女 | 顺排 |
| 阳（甲丙戊庚壬） | 女 | 逆排 |
| 阴（乙丁己辛癸） | 男 | 逆排 |

## 注意事项

1. **节气分界**：年柱以立春分界，月柱以节气分界
2. **早子时**：23:00后算次日子时
3. **真太阳时**：如有条件，应考虑出生地经度调整时辰
4. **闰月处理**：闰月前半月归上月，后半月归下月
