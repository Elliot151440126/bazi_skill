# 八字命理分析 Skill

---

## ⚠️ CRITICAL BEHAVIOR RULES

### 1. 分析顺序（铁律）
**必须严格遵循以下顺序，不得跳过或调换：**
```
排盘 → 天干地支互动 → 原局（先天格局）→ 前事验证（⚠️ 必须用户确认）→ 大运（十年方向）→ 流年（年份吉凶）→ 流月（月份应期）
```
- ❌ 禁止在未分析原局的情况下直接做大运/流年分析
- ❌ 禁止跳过前事验证直接做深度分析
- ❌ 禁止跳过大运直接看流年
- ❌ 禁止跳过流年直接看流月

### 2. 信息收集规则
- ✅ 必须先确认：出生年、月、日、时、性别
- ✅ 时辰不确定时，列出可能的两个时辰供用户选择
- ❌ 不要假设或编造出生信息

### 3. 输出规范
- ✅ 每一步分析都要有明确结论
- ✅ 用表格呈现五行力量、十神分布
- ✅ 最终给出用神、忌神、喜神、仇神
- ❌ 不要模棱两可的表述

## 🔍 Phase 2: 天干地支互动分析

### STEP 4: 分析天干地支互动

**⚠️ 此步骤为必经步骤，不得跳过！** 在分析原局格局之前，必须先穷举四柱中所有天干地支的互动关系。这些关系直接影响五行力量的消长、格局的成败、用神的取舍。

**分析顺序：天干互动 → 地支互动 → 天干地支交叉互动**

---

#### 4.1 天干互动

**天干五合：**

| 组合 | 化 | 条件 |
|------|-----|------|
| 甲己 | 合化土 | 辰戌丑未月或四柱土旺 |
| 乙庚 | 合化金 | 申酉月或四柱金旺 |
| 丙辛 | 合化水 | 亥子月或四柱水旺 |
| 丁壬 | 合化木 | 寅卯月或四柱木旺 |
| 戊癸 | 合化火 | 巳午月或四柱火旺 |

**合的几种状态：**
- **合化成功**：化神当令且旺，两干失去原五行，变为化出的五行
- **合而不化**：化神不当令或不够旺，两干牵绊但保留原五行（力量减弱）
- **合中带克**：如丙辛合，辛金被丙火克（有合之名，有克之实）

**天干相冲：**

| 组合 | 五行 | 冲的结果 |
|------|------|----------|
| 甲庚冲 | 木↔金 | 金克木，甲受伤（庚旺时）或庚受损（甲旺时） |
| 乙辛冲 | 木↔金 | 同上 |
| 丙壬冲 | 火↔水 | 水克火，丙受伤（壬旺时）或壬受损（丙旺时） |
| 丁癸冲 | 火↔水 | 同上 |

**冲的判断原则：**
- 旺者冲衰，衰者受伤
- 得令、得势、得地者胜
- 冲主动荡、变化、分离

#### 4.3 双重关系与特殊组合

**⚠️ 同一组地支可能同时具有两种关系，必须逐一排查，不可遗漏！**

**一、一身兼两职的地支组合：**

| 组合 | 关系A | 关系B | 判断规则 |
|------|-------|-------|----------|
| 巳申 | 六合（合化水） | 三刑（寅巳申） | 有寅→三刑破合；无论合 |
| 寅亥 | 六合（合化木） | 相破 | 合中带破，先好后坏，关系不纯粹 |
| 丑未 | 六冲 | 三刑（丑戌未） | 有戌→三刑；无论冲 |
| 卯午 | 相破 | 暗合（木火相生） | 主流论破，次论相生 |
| 子酉 | 相破 | — | 金水相破，暗耗之象 |
| 丑戌 | 三刑（丑戌未） | — | 有未则三刑齐全 |
| 辰戌 | 六冲 | — | 同属土冲力大，冲中带刑意 |

**二、合冲并存的判断：**

| 场景 | 例子 | 判断原则 |
|------|------|----------|
| 六合遇冲 | 子丑合，午来冲子 | 冲能破合，合被冲开 |
| 六冲遇合 | 子午冲，丑来合子 | 合能解冲，子被丑绊住不再冲午 |
| 三合遇冲 | 申子辰合水局，午冲子 | 冲中神（子），三合局散 |
| 三会遇冲 | 寅卯辰会木，申冲寅 | 冲首支（寅），三会局受损 |
| 三合遇三合 | 申子辰（水）+ 寅午戌（火） | 水火对冲，两局俱伤 |
| 三会遇三会 | 寅卯辰（木）+ 申酉戌（金） | 金木对冲，两局俱伤 |

**三、合冲刑害破并存时的力量排序：**

```
三会局 > 三合局 > 六合 > 六冲 > 三刑 > 六害 > 相破
```

**但需注意：**
- 得令（当月旺）者力量加成，可逆袭排名
- 三刑齐全（如寅巳申、丑戌未）时，刑的力量可超过普通六合
- 冲中神（三合局的中间支）可直接破局

**四、特殊组合速查：**

| 组合 | 名称 | 含义 |
|------|------|------|
| 天合地合 | 如甲子见己丑（甲己合+子丑合） | 上下同心，关系极紧密 |
| 天克地冲 | 如甲子见庚午（甲庚冲+子午冲） | 上下皆冲，动荡最大 |
| 天克地合 | 如甲子见己午（甲己合+子午冲） | 天干合、地支冲，貌合神离 |
| 天冲地合 | 如甲子见庚丑（甲庚冲+子丑合） | 天干冲、地支合，表面不合暗中合作 |

#### 4.4 分析方法

**逐柱穷举法（必须按此执行）：**

1. **天干层面**：列出四柱天干，逐一检查是否存在五合、相冲
2. **地支层面**：列出四柱地支，逐一检查是否存在六合、六冲、三合、三会、相刑、六害、相破
3. **干支交叉**：检查天干克地支、地支克天干的关系（如甲申——甲木坐申金，截脚之象）

**⚠️ 注意事项：**
- 合冲并存时，合能解冲，冲能破合——需判断哪个力量更大
- 刑害并存时，刑的力量大于害
- 三合、三会成局时，该局五行力量大增，需重新评估旺衰
- 天干合化成功后，原五行消失，十神也要相应改变

## 🔍 Phase 3: 原局分析（先天格局）

### STEP 5: 判断日主旺衰

**核心要素：**

#### 5.1 看月令（最重要，占40-50%力量）
- **得令**：日干五行与月支五行相同或被月支生扶
- **失令**：日干五行被月支克泄耗

**月令旺相休囚死：**
| 月令状态 | 含义 | 力量 |
|----------|------|------|
| 旺 | 当令，最旺 | 100% |
| 相 | 得生，次旺 | 80% |
| 休 | 休息，中平 | 50% |
| 囚 | 被克，衰弱 | 30% |
| 死 | 最弱 | 10% |

#### 5.2 看通根（地支根气）
- **本气根**：地支藏干与日干相同（力量最强）
- **中气根**：地支藏干与日干相生（力量次之）
- **余气根**：地支藏干与日干相克（力量最弱）

**地支藏干表：**
| 地支 | 本气 | 中气 | 余气 |
|------|------|------|------|
| 子 | 癸 | - | - |
| 丑 | 己 | 癸 | 辛 |

## 🔍 Phase 4: 前事验证

### STEP 8: 前三年流年推断 + 用户验证

**⚠️ 此步骤为必经步骤，不得跳过！**

在完成原局分析（旺衰、格局、用神）之后，先用过去三年的流年做一次「前事推断」，供用户验证准确度。**确认准了再继续后面的深度分析，不准则调整方向。**

**为什么要这样做：**
- 八字理论存在多种流派和解读角度，同一八字不同人可能有不同判断
- 前事验证是最直接的校准手段——用事实说话
- 避免在错误方向上浪费大量分析精力

---

#### 8.1 分析范围

**时间范围：** 今年 + 前两年（共 3 年）

**⚠️ 注意跨大运：** 如果这 3 年恰好跨了两步大运交界，必须分别说明：
- 旧大运尾声的影响
- 新大运初期的冲击
- 交接年份往往动荡最大

**⚠️ 注意应象广度：** 同一个十神/五行变动，可能应验在不同领域，不要自己锁定一个方向。应给出多个可能的应象，让用户自己对应。

---

**逐年分析，每一年都要检查：**

#### 8.4 用户反馈后的处理

**情况一：用户确认「大部分准」**
```
很好，分析方向基本正确。接下来我继续深入分析大运、流年和流月。
```
→ 继续 STEP 9 起的完整分析流程。

**情况二：用户说「不太准」或「部分不准」**
```
前事有出入，说明我对这个八字的某些判断需要修正。
为了更准确地分析，能否告诉我：
1. 过去 3 年中，你觉得最重要的 1-2 件事情是什么？（比如：结婚、升职、搬家、生病、破财等）

有了真实案例，我可以重新校准旺衰、格局和用神的判断。
```
→ 根据用户提供的真实事件，重新审视：
  - 日主旺衰判断是否需要调整
  - 格局是否判断有误
  - 用神忌神是否取反
  - 大运交接的影响是否被低估
→ 修正后再给出新的前事推断，再次验证。

**情况三：用户拒绝提供信息**
```
没问题，我按照当前的判断继续分析，你可以参考后面的结论自行对照。
```
→ 继续 STEP 9，但注明「未经前事校准，仅供参考」。

## 🔍 Phase 6: 流年分析（年份吉凶）

### STEP 11: 分析当前流年

**流年分析要点：**
